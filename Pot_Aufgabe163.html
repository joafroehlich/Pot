200 * ex = 2000 

Logarithmieren: 

ln(200 * ex) = ln2000   

ln200 + lnex = ln2000   

ln200 + x * lne = ln2000    

ln200 + x = ln2000   |-ln200 

x = ln2000 – ln200 = 7,6009 – 5,2983 = 2,3

lnex = ln10
