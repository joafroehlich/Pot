Aufgabe 237

Ein Bienenvolk ist innerhalb einer Woche von 50 000 Bienen 
auf 40 000 zurückgegangen. 
Nach wie viel Wochen ist die Population bei exponentieller 
Abnahme nur noch halb so groß?  
In welcher Woche stirbt die letzte Biene?  
 
B1 = B0 * q 
 
40 000 = 50 000 * q |:50 000 
 
       40 000  
q = ---------- = 0,8 = Abnahmefaktor 
       50 000  
 
Bn = B0 * qn 
 
0,5 = qn 
 
Logarithmieren: 
 
lg 0,5 = lg 0,8n 
 
lg 0,5 = n * lg 0,8 |:lg 0,8 
 
     lg 0,5       - 0,301 
n = --------- = ----------- = 3,1 Wochen 
     lg 0,8      - 0,0969 
 
Bn = B0 * qn 
 
1 = 50 000 * 0,8n |:50 000 
 
    1 
---------- = 0,8n 
  50 000 
 
Logarithmieren: 
 
         1 
lg ---------- = lg 0,8n 
     50 000 

lg 1 – lg 50 000 = n * lg 0,8n |:lg 0,8 
 
      lg 1 – lg 50 000      0 – 4,699  
n = ------------------- = ------------- = 48,5 Wochen 
          lg 0,8            - 0,0969 
 
In der 49. Woche gibt es keine Bienen mehr.
