5x = 2 * 7x+1  

Gleichung logarithmiert: 

lg5x = lg(2 * 7x+1) 

x * lg5 = lg2 + lg7x+1 

x * lg5 = lg2 + (x + 1) * lg7 

x * lg5 = lg2 + x * lg7 + lg7 | -x * lg7 

x * lg5 – x * lg7 = lg2 + lg7 

x * (lg5 – lg7) = lg2 + lg7 | :( lg5 – lg7)
